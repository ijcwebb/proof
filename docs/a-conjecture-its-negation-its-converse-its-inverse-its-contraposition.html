15.1 \(\Rightarrow\) revisited

We discussed the symbol “\(\Rightarrow\)” in Chapter 9. We return to it here, clarifying its meaning a little further.

For sets \(A\) and \(B\), \(A \leftarrow B\) means “all elements in \(A\) go to \(B\).”

Consider the (empty) conjecture map below:

The conjectures we will meet in this course are generally of the form \(P \Rightarrow Q\) (e.g. “The product of three consecutive integers is divisible by 6,” “For any negative number \(a\), \(a^2 >a\),” “All primes are odd”). In other words:

  • \(P \Rightarrow Q\): ALL elements in \(P\) go to \(Q\) (and never \(T\)).

Some other possible relationships within this map are:

  • \(P \Rightarrow T\): ALL elements in \(P\) go to \(T\) (and never \(Q\)).
  • \(S \Rightarrow Q\): ALL elements in \(S\) go to \(Q\) (and never \(T\)).
  • \(S \Rightarrow T\): ALL elements in \(S\) go to \(T\) (and never \(Q\)).

Also,the following relationships are possible, but different, from those four above:

  • \(Q \Rightarrow P\): ALL elements in \(Q\) came from \(P\) (i.e. no elements from \(S\) went to \(Q\)).
  • \(T \Rightarrow P\): ALL elements in \(T\) came from \(P\) (i.e. no elements from \(S\) went to \(T\)).
  • \(Q \Rightarrow S\): ALL elements in \(Q\) came from \(S\) (i.e. no elements from \(P\) went to \(Q\)).
  • \(T \Rightarrow S\): ALL elements in \(T\) came from \(S\) (i.e. no elements from \(P\) went to \(T\)).

Exercise 15.1 Take a look at the following conjecture map (from Chapter 8). Which of the eight relationships from above are true in this map?

Solution:

  • \(P \Rightarrow Q\) is true in this map: An even integer multiplied by any integer remains even.

  • \(P \Rightarrow T\) is false in this map (as \(P \Rightarrow Q\)).

  • \(S \Rightarrow Q\) is false in this map. Sometimes an odd intger multiplied by another integer is even (if the second integer is even) but not always. We can’t use \(\Rightarrow\) unless every element of \(S\) goes to \(Q\).

  • \(S \Rightarrow T\): is false in this map for the same reason \(S \Rightarrow Q\) is false. Sometimes an odd integer multiplied by another integer is odd (if the second integer is odd) but not always. We can’t use \(\Rightarrow\) unless every element of \(S\) goes to \(T\).

  • \(Q \Rightarrow P\) is false in this map: not all elements in \(Q\) come from \(P\) - some come from \(S\).

  • \(T \Rightarrow P\) is false in this map: in fact, no elements in \(T\) come from \(P\).

  • \(Q \Rightarrow S\) is false in this map: some elements in \(Q\) come from \(S\) but some come from \(P\).

  • \(T \Rightarrow S\) is true in this map: Any element in \(T\) definitely came from \(S\) (i.e. none came from \(P\)).

These four relationships have subtle distinctions, and you should try to make sure you are happy with the difference between them. Do you understand why \(P \Rightarrow Q\) is different to \(Q \Leftarrow P\), for instance? To test yourself, attempt the following exercise.
